The BBX Super Factory is a high-density building automation gateway engineered for intensive data environments.

Equipped with a High-Performance Quad-Core Edge Engine, the Super Factory delivers enterprise-grade processing power to the network edge, enabling real-time ad-hoc analytics and high-frequency data trending.
